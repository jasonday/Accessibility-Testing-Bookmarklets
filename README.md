# Accessibility testing bookmarklets that work with the Shadow DOM
[Get the bookmarklets](https://jasonday.github.io/Accessibility-Testing-Bookmarklets/)


## Bookmarklets
- aName - Shows the various pieces for accessible name calculation; includes state, role, element, etc.
    - Based on [lloydi/WTFocus](https://github.com/lloydi/WTFocus) but then stripped down and heavily modified for shadow DOM and other needs. 
- aImage - Shows the text alternatives, image type, and text alternative reference for all image types
- aStructure - Highlights the structural elements in the page
